# 💎 Jewellery Sales Dashboard

## 📌 Project Overview

The **Jewellery Sales Dashboard** is an interactive business intelligence dashboard developed using **Microsoft Power BI**. The project analyzes jewellery sales data and provides valuable insights into revenue, product performance, salespersons, stock quantity, and city-wise sales.

The dataset was cleaned and prepared using **Microsoft Excel** before being imported into Power BI for visualization.

---

## 🎯 Project Objectives

- Analyze overall jewellery sales performance.
- Track total revenue and orders.
- Identify top-selling jewellery products.
- Compare category-wise sales.
- Analyze salesperson performance.
- Monitor city-wise revenue distribution.
- Support business decisions with interactive visualizations.
